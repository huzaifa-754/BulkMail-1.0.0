BULK MAIL TOOL - INSTRUCTIONS

Option: 1 (With Java)

1. Unzip then place DLL file from lib to your system32 folder as per architecture 32 or 64 bit
2. Check sample file in Excel folder, review attachments and signature logo path as per yours
3. It takes criteria from sheet1 C and then filter sheet2 "D" column for mail body table
4. Click on exe or jar
5. Select Excel file when prompted

Option: 2 (With VBA)
1. Open Sample Excel file in Excel folder
2. Setup your date in Sheet 1-3, It takes Criteria from Sheet1 "C" and make mail body table by filtering Sheet2 "D"
3. All set "Click on Submit"


Requirements:

* Windows OS
* Microsoft Outlook installed and logged in

Notes:


* Do not delete lib folder (required for Outlook connection)
* Keep logos inside resources folder
